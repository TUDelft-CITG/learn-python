Exercise 5.2.4#

The file arr.txt contains \(1000\) randomly chosen \(x,y\) locations of points; both \(x\) and \(y\) vary between \(-10\) and \(10\). Load the data using np.loadtxt(), and store the first row of the array in an array called x and the second row in an array called y. First, plot a red dot for all points. Then, on the same graph, plot a blue dot for all \(x,y\) points where \(x<-2\) and \(-5\le y \le 0\). Finally, plot a green dot for any point that lies in the circle with center \((x_c,y_c)=(2.5,0)\) and with radius \(R=5.0\).

Hint: it may be useful to compute a new array for the radial distance \(r\) between any point and the center of the circle using the formula \(r=\sqrt{(x-x_c)^2+(y-y_c)^2}\). Lastly, use the plt.axis(‘equal’) command to make sure the scales along the two axes are equal and the circular green area looks like a circle.

# your code here
...

###BEGIN SOLUTION TEMPLATE=
file = np.loadtxt(path + 'arr.txt')
x = file[0]
y = file[1]

plt.plot(x, y, 'r.')

mask_blue = (x < -2) & (y >= -5) & (y <= 0)
plt.plot(x[mask_blue], y[mask_blue], 'b.')

x_c, y_c = 2.5, 0
